Todolist is an exercise who's running Node.js, Express.js, MongoDB and Angular.

## Install

```sh
npm install
```
Install MongoDb https://doc.ubuntu-fr.org/mongodb

## Development server

```sh
sudo sudo systemctl start mongod
npm start 
ngserve
```


Open the project in your web browser
